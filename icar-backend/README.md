### What is it?
<b>iCar App</b>

### Technologies used: 
- [Node](https://nodejs.org/en/)

### How to run?
This project uses NodeJS, so in order for it to work, frist navigate insde each project folder and run npm install to download the proper dependencies: 
```
cd icar-backend
npm install
npm run start
```
